# Jasmine J. Cash Portfolio

Static portfolio website for Jasmine J. Cash, PhD, CSCS, highlighting her research, projects, publications, presentations, teaching, and recognition.

## Pages

- `index.html` - Home and research overview
- `projects.html` - Research projects organized by theme
- `expertise.html` - Research expertise, tools, credentials, and certifications
- `publications.html` - Selected peer-reviewed publications
- `presentations.html` - Selected conference and invited presentations
- `teaching.html` - Teaching and mentorship
- `recognition.html` - Grants, honors, service, and professional contributions
- `404.html` - GitHub Pages fallback page
